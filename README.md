# s3-static-web
Turn your bucket into a webpage without the lambda hassle


# Configure

The app requires a number of environment variables:

* AWS_ACCESS_KEY_ID
* AWS_SECRET_ACCESS_KEY
* AWS_REGION
* *_Optional_*
  * BASIC_USER 
  * BASIC_PASS
